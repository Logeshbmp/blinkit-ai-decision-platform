# Blinkit AI Decision Platform

An end-to-end AI + Data Analytics platform built using:

- PostgreSQL
- Streamlit
- Machine Learning
- RAG (Retrieval Augmented Generation)

## Features

### 1 Analytics Dashboard
- Revenue KPI
- ROAS
- Delivery Delay
- Profit

### 2 Delivery Risk Calculator
ML model predicting delivery delay probability.

### 3 AI Business Assistant
LLM powered assistant answering WHY questions using customer feedback.

## Tech Stack

Python  
PostgreSQL  
Streamlit  
Scikit-learn  
LangChain  
FAISS

pip install -r requirements.txt
